Honey I think I... found the Movie

This app uses the GuideBox API (www.guidebox.com) to enable potential users to search for content they are looking to watch. It returns the various platforms a movie can be retrieved from (rental, purchase or streaming). The app is currently limited to searching for movies, but the API can be filtered by movies, shows, people, etc... These capabilities can be added at a later stage. The app is designed to be fully responsive across mobile, tablet and desktop screen resolutions.

Welcome to the Movie Search

Tired of the long searches across all apps and platforms for the movie of the night?

Simply enter the movie title you are thinking of in the search box.

The app will return all the matching titles we can find so you can select the correct one.


And which platform?

Next, select the movie of your choice, and we will let you know the platforms it can be streamed from, the format (SD or HD), and whether it can be viewed for free, rented or purchased.
